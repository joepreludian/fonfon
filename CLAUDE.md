# Fonfon

Opinionated VPS Configurator tool.
It setups a base system in a Linux environment to be ready to serve applications. It includes SSH Hardening, package installation like docker, tailscale and SDCI and setup services in order to have a simple system. The name "Fonfon" is an omage for my cat Persephone. She is a orange one with a short tail. =)

## Architecture

Fonfon is a python-cli application, self-contained in a `pex` executable for quick run into a server, without dependencies. It uses `click` and `rich` as the main libraries. Also will be responsible for managing server issuing commands and so on.
