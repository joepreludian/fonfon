# Fonfon

An opinionated VPS configurator.

Fonfon prepares a fresh Linux server to serve applications, applying sensible
defaults so you don't have to wire everything by hand:

- **SSH hardening** — locks down remote access out of the box.
- **Core packages** — installs Docker, Tailscale, and SDCI.
- **Service setup** — configures the services needed for a simple, ready system.

It ships as a self-contained executable, so it runs on a bare server with no
dependencies to install first.

> The name "Fonfon" is an homage to my cat Persephone — an orange one with a
> short tail. 🐱

## ⚠️ Alpha software — not for production

Fonfon is in **alpha**. It is under active development, its behavior may change
without notice, and **production use is not recommended**. Stick to disposable
or test servers while it stabilizes.
